Where are my run files?
They live in your Slay the Spire 2 save folder, typically:
C:\Users\<you>\AppData\Roaming\SlayTheSpire2\steam\<steam-id>\profile1\saves\history
Every completed run is a .run file in that folder. Use Select history folder and point it there, or drag the .run files in directly.
